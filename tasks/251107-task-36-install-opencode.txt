TASK-36: Install OpenCode AI Coding Agent in Docker Container

OBJECTIVE:
Install OpenCode (https://github.com/sst/opencode) - a 100% open-source AI coding agent for the terminal - in the Hephaestus Docker container.

PRIORITY: HIGH

BACKGROUND:
OpenCode is "The AI coding agent built for the terminal."
- 100% open-source alternative to Claude Code
- Provider-agnostic (Anthropic, OpenAI, Google, local models)
- Built-in Language Server Protocol (LSP) support
- Client/server architecture for remote operation
- Terminal-focused UI (TUI)

Repository: https://github.com/sst/opencode
Documentation: https://opencode.ai/docs

═══════════════════════════════════════════════════════════════════
INSTALLATION METHODS
═══════════════════════════════════════════════════════════════════

**Option 1: Quick Install Script (Recommended for Docker)**

```bash
curl -fsSL https://opencode.ai/install | bash
```

**Option 2: npm/pnpm/yarn/bun**

```bash
npm i -g opencode-ai@latest
```

Or:
```bash
pnpm add -g opencode-ai@latest
bun add -g opencode-ai@latest
yarn global add opencode-ai@latest
```

**Option 3: Homebrew (macOS/Linux)**

```bash
brew install opencode
```

═══════════════════════════════════════════════════════════════════
DOCKER INSTALLATION (DOCKERFILE)
═══════════════════════════════════════════════════════════════════

Add to docker-hephaestus/Dockerfile.new:

**Method 1: Quick Install Script**
```dockerfile
# Install OpenCode via official installer
RUN curl -fsSL https://opencode.ai/install | bash

# Add OpenCode to PATH
ENV PATH="/root/.opencode/bin:${PATH}"
```

**Method 2: npm Install**
```dockerfile
# Install OpenCode via npm
RUN npm i -g opencode-ai@latest
```

Note: Check if Node.js/npm is already installed in Dockerfile. If not, add:
```dockerfile
# Install Node.js and npm (if not already present)
RUN curl -fsSL https://deb.nodesource.com/setup_20.x | bash - && \
    apt-get install -y nodejs

# Then install OpenCode
RUN npm i -g opencode-ai@latest
```

═══════════════════════════════════════════════════════════════════
INSTALLATION IN RUNNING CONTAINER (QUICK TEST)
═══════════════════════════════════════════════════════════════════

1. SSH into container:
   ```bash
   ssh user@localhost -p 2224
   ```

2. Install OpenCode (Quick Install):
   ```bash
   curl -fsSL https://opencode.ai/install | bash
   ```

3. Add to PATH (if needed):
   ```bash
   echo 'export PATH="$HOME/.opencode/bin:$PATH"' >> ~/.bashrc
   source ~/.bashrc
   ```

4. Or install via npm (if Node.js is available):
   ```bash
   npm i -g opencode-ai@latest
   ```

═══════════════════════════════════════════════════════════════════
IMPORTANT NOTES
═══════════════════════════════════════════════════════════════════

- **Remove old versions**: If version < 0.1.x exists, remove before upgrading
- **Custom install directory**: Use environment variables:
  - `$OPENCODE_INSTALL_DIR`
  - `$XDG_BIN_DIR`
  - `$HOME/bin`
  - Default: `$HOME/.opencode/bin`
